== Description ==
in wordpress website hack plugins are 90% responsible. its hard to tell which pluign is Vulnerable. 
No its Not!  install   WP Vulnerabilty Scanner and you are one click away to find out which of the pluign you is/are using is Vulnerable.

AND IT’S 100% FREE!

this pluign is using WPScan Vulnerability Database to check your installed  plugins for any publicly known  Vulnerabilities. 
Thanks to  WPScan Vulnerability Database[https://wpvulndb.com]

== Installation ==
FROM YOUR WORDPRESS DASHBOARD

1. Visit ‘Plugins > Add New’
2. Search for ‘WP Vulnerabilty Scanner’
3. Activate WP Vulnerabilty Scanner from your Plugins page. 
4. All set start scanning 

FROM WORDPRESS.ORG

1. Download WP Vulnerabilty Scanner
2. Upload the ‘WP Vulnerabilty Scanner’ directory to your ‘/wp-content/plugins/’ directory, using your favorite method (ftp, sftp, scp, etc…)
3. Activate WP Vulnerabilty Scanner from your Plugins page. 
4. All set start scanning
